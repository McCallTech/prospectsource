# Prospect Source 


## Getting up and running: 

### Hipster one-liner:
```
curl -0 https://raw.githubusercontent.com/McCallTech/prospectsource/master/goScript | bash
```

### Copy Pasta:
```
git clone https://github.com/McCallTech/prospectsource.git && \
cd todo-react-redux && \
nvm i 6 && \
nvm use 6 && \
npm i
```
